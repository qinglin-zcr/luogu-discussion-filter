# Luogu Discussion Filter

用于屏蔽洛谷讨论区特定内容的篡改猴脚本。

---

# 安装

## 安装 Tampermonkey

从浏览器扩展商店安装 Tampermonkey。

---

## 安装脚本

安装后：

1. 新建脚本
2. 粘贴本仓库中的脚本代码
3. 保存

---
